hyperion.sqlite
============

A Hyperion implementation for SQLite3

## Types

### Natively Supported

* `java.lang.Integer`

### Supported by Packer/Unpacker

* `java.lang.Boolean`
* `java.lang.Float` as column type `DOUBLE` or `FLOAT`
