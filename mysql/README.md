hyperion.mysql
============

A Hyperion implementation for MySQL

## Types

### Natively Supported

* `java.lang.Boolean`
* `java.lang.Integer`

### Supported by Packer/Unpacker

* `java.lang.Float` as column type `DOUBLE` (`FLOAT` will truncate)
